# SNP Skill 新手入门

你不需要先学会 Skill、Prompt、RAG 或工作流。

先带着一个真实的护理问题来。SNP Skill 会先判断你现在处在哪一步，再帮你把问题推进到一个可以完成、可以核实的下一步。

## 先记住一句话

> 不用先研究工具。把你现在最真实的问题交出来，先只推进当前一步。

## 你可以拿它做什么

- 判断自己或孩子是否适合认真了解护理
- 比较国内专科、本科、中外合作和海外护理路径
- 梳理英语、实习、升学、执照、岗位和薪资准备
- 初筛澳洲 OBA、Ahpra/NMBA、日本护士、介护/SSW 等方向
- 复核学校、费用、岗位、机构话术和政策信息
- 把已经选定的方向拆成今天可以完成的一件事

它不替你做最终志愿决定，也不预测录取，不承诺就业、签证、执照或移民结果。

## 第一次使用：手机 AI

这是最适合普通用户的入口。

### 第 1 步：下载 Lite 文件

打开 [`Lite 中文单文件`](../dist/shawn-nursing-pathway-lite.md)，下载它，或者打开后复制全文。

也可以直接使用固定地址：

<https://raw.githubusercontent.com/jc99110901-tech/shawn-nursing-pathway/main/dist/shawn-nursing-pathway-lite.md>

Lite 文件就是给普通 AI 对话使用的简化版规则。第一次不需要下载完整套件，也不需要安装全部模块。

### 第 2 步：放进新的 AI 对话

在当前支持文件上传的豆包、千问或其他 AI 对话中，上传 Lite 文件。

如果手机当前不能上传 `.md` 文件：

1. 打开 Lite 文件。
2. 复制全部文字。
3. 粘贴到一个新的 AI 对话。

平台的按钮名称和文件能力会更新，以你当前手机显示的功能为准。

### 第 3 步：发送启动语

上传后，先发送这句话：

```text
请把我上传的 SNP Skill 文件作为本次对话规则。先判断我的问题属于哪一类；缺少关键信息时只问最少的问题。不要替我做最终决定。方向已经选定时，每次只给我一个可以完成并带回结果的下一步。
```

然后直接说你的问题，不需要把问题改写得很专业。

例如：

```text
孩子高考分数一般，家里觉得护理稳定，但孩子不太愿意。先帮我们做护理适配初筛。
```

```text
我是护理本科大二，英语一般，想看看以后有哪些收入更高的路径，我现在先做什么？
```

```text
我是国内注册护士，想通过澳洲 OBA 去做 RN，第一步该核实什么？
```

```text
我不想一直做临床，想了解国际医院、护理教育、医疗器械和健康科技岗位的差别。
```

## 你会看到什么

一个正常的回答通常会先：

1. 识别你的阶段和目标。
2. 补问少量真正影响判断的信息。
3. 判断当前更需要做适配、路径比较、规划、学习、岗位准备或信息核验。
4. 给出可能的方向、主要风险和下一步问题。
5. 让你带回一个结果，再继续下一轮。

它不会一上来给你一长串学校、国家或全年计划。

## SNP Skill 怎样继续使用

你可以把它理解成一条主路由：

```text
真实问题
   ↓
判断当前阶段
   ↓
选择最相关的护理模块
   ↓
完成一个最小任务
   ↓
带回结果，继续下一步
```

你不需要记住模块名称。只要继续说：

```text
我完成了刚才的任务，这是我的结果。请根据这个结果给我下一步唯一要做的事。
```

## 方向已经选定以后

如果你已经暂定一个方向，不要让 AI 一次列完整人生计划，可以这样问：

```text
我已经暂定澳洲方向。先给我今天唯一要做的一个任务，完成后我把结果带回来。
```

学习英语或护理知识时，也按一课一反馈使用：

```text
我想学国际医院护士交班英语。先测试我的水平，只给第一课，不要一次生成整套课程。
```

## 换平台怎么选文件

第一次不要同时安装所有版本。根据你使用的地方选择一个入口：

| 你在哪里使用 | 先用什么 |
|---|---|
| 手机豆包、千问或普通 AI 对话 | Lite 中文单文件 |
| 扣子、Dify、FastGPT、MaxKB、百炼等知识库平台 | Lite 做角色指令，Full 做知识库 |
| WorkBuddy / 腾讯云智能体开发平台 | WorkBuddy Skill ZIP |
| Codex | Suite ZIP 或 `skills/` 完整套件 |

详细平台说明见 [`QUICKSTART.md`](../QUICKSTART.md) 和 [`1 分钟上手指南 PDF`](SNP-Skill-1分钟上手指南.pdf)。

不同平台的系统提示词、文件上传、知识库、工作流和长期记忆能力不同。本仓库不承诺所有平台都能一键导入。

## Codex 用户

下载 [`Suite ZIP`](../dist/shawn-nursing-pathway-suite.zip)，解压后把其中 `skills/` 下的 8 个 Skill 文件夹复制到 Codex 的 skills 目录。

安装完成后，在新任务中输入：

```text
$shawn-nursing-pathway 我是护理本科大二，英语一般，想看看以后有哪些收入更高的路径，我现在先做什么？
```

普通用户只需要记住主入口，不需要记住 7 个细分 Skill 的名字。

## 每次使用前，最好带上这些信息

不需要一次写全，能提供多少写多少：

- 当前阶段：高考生、家长、在读护理学生、国内护士或转行者
- 学历和注册状态
- 大致成绩或专业层次，如愿意提供
- 英语能力
- 预算和时间
- 是否能接受夜班、轮班和护理实操
- 是否接受海外生活
- 最终更重视就业、留学、海外注册、长期居留或家庭稳定

不要上传身份证号、准考证号、护照号、完整成绩截图、银行信息、未脱敏合同或私人医疗记录。

## 如果回答不够好

先检查这 4 件事：

1. Lite 文件是否真的上传或复制完整。
2. 是否在新对话中先发送启动语。
3. 问题是否包含当前阶段和真实目标。
4. 涉及学校、政策、工资或费用时，是否要求标明年份、来源和核验日期。

然后把问题缩小：

```text
先不要比较所有国家。只比较日本护士和日本介护的入口差异，并列出我需要核实的官方问题。
```

## 以后怎样更新

GitHub `main` 分支是公开发行真源。你下载后上传到平台的文件属于版本快照；仓库更新后，需要重新下载并替换旧文件。

当前公开文件和版本信息见 [`release.json`](../release.json) 与 [`dist/manifest.json`](../dist/manifest.json)。

不要把旧回答当成当前政策。涉及学费、岗位、工资、执照、签证和移民问题时，要求 SNP Skill 说明数据日期、来源状态，并优先核验最新官方文件。查不到当前版本时，应直接说没有找到。

## 边界提醒

SNP Skill 做的是公开信息整理、路径解释、护理适配度初筛、风险提示和问题清单。

它不：

- 预测录取结果
- 输出最终志愿排序
- 承诺录取、毕业、就业、个人薪资、签证、执照或移民
- 推荐或背书学校、中介、培训机构、雇主或移民代理
- 代表考试院、学校、护理监管机构、雇主或任何官方渠道

真正涉及录取、注册、签证、就业和移民时，以相关官方最新文件为准。

## 一张 SNP 进度卡

跨平台或隔一段时间继续时，可以把这张卡带回对话：

```text
SNP 进度卡

当前阶段：
学历／注册状态：
目标方向：
已经确认：
还没有确认：
本轮完成的任务：
带回的结果：
下一步只做：
```

开始时，你只需要写一句：

```text
这是我的 SNP 进度卡，请先判断我现在最应该推进哪一步，只给一个下一步任务。
```
