# 案例 3：日本护理和介护有什么区别

![日本护理和介护路径区分案例](../assets/case-cards/case-03-japan-nursing-caregiving.svg)

## 用户原话

```text
日本护理和介护有什么区别？是不是先去介护就能转护士？
```

## Skill 路由

进入：

- 日本护理/介护路径区分
- 语言门槛核验
- 角色、考试、签证和长期路径拆解

不直接进入：

- 日本就业承诺
- 转护士承诺
- 移民承诺

## 建议回答

```markdown
## 边界提醒
我可以帮你解释日本护理和介护路径的区别，以及需要核实的官方条件，但不能承诺就业、签证、执照或长期居留结果。

## 初步判断
日本“护士”和“介护”不是同一个职业路径。

护理更接近护士资格和医疗护理体系，通常涉及更强的专业资格、考试和语言要求。介护更偏长期照护、老人照护和生活支持，可能涉及特定技能、介护技能测试、日语要求和雇主路径。

## 主要风险
- 不能把“先去介护”简单等同于“以后一定能转护士”。
- 日语不是附加项，而是路径能不能走下去的核心门槛。
- 学校、考试、雇主、签证和长期居留是不同问题。

## 下一步要核实
- 你说的是护士、介护士、介护福祉士，还是特定技能介护。
- 当前学历和护理背景是否支持护士路径。
- 日语目前水平和可接受学习周期。
- 目标项目是否有官方学校、雇主、考试和签证依据。

## 下一步建议
- 先区分护士路径和介护路径。
- 再评估日语学习周期。
- 最后核验学校/雇主/签证文件。
```

## English Summary

The skill should separate the nurse route from the caregiving route. It should make language, role definition, examination, employer, visa, and residence layers explicit, and it should not imply that caregiving automatically converts into a nursing license.
