# 案例 4：中国护士问澳洲 OBA / Ahpra / NMBA 注册路径

![中国护士澳洲 OBA 初筛案例](../assets/case-cards/case-05-australia-oba-iqnm.svg)

## 用户原话

```text
我是国内注册护士/护理本科毕业，想通过 OBA 去澳洲做 RN。这个路径能不能走？要不要先报机构？
```

## Skill 路由

进入：

- 澳洲 OBA/IQNM 初筛
- 国内护士/护理毕业生用户分层
- Ahpra/NMBA Self-check、Stream A/B/C、Portfolio、MCQ/NCLEX-RN、OSCE、Registration 层拆解
- ANMAC、签证、就业和长期居留分层核验
- 机构话术核验

不直接进入：

- 注册结果判断
- 考试通过承诺
- 就业、签证、PR 或移民承诺
- 培训机构、中介、学校、雇主或移民代理推荐

## 建议回答

```markdown
## 边界提醒
我可以帮你把澳洲 Ahpra/NMBA 的国际护士注册路径拆开做初筛，但不能承诺注册、NCLEX-RN、OSCE、ANMAC 评估、工作、签证、PR 或移民结果，也不推荐任何培训机构、中介、学校、雇主或移民代理。当前规则以 Ahpra/NMBA/ANMAC 和澳洲官方最新文件为准。

## 用户画像
现在最需要先确认这几项：
- 你最高护理学历是什么？中专/大专/本科/硕士？
- 你是否已经取得中国护士资格和护士执业注册？目前是否有效？
- 你是否有可追溯的毕业证、学位证、成绩单、课程/实习或临床小时证明？
- 你有几年临床经验？科室、近期执业和推荐证明能否对应？
- 英语目前到什么水平？目标是 IELTS/OET/PTE 还是其他证明路径？
- 是否能接受去澳洲参加 OSCE，以及机票、住宿、签证、重考等现金流风险？
- 你的最终目标是注册、就业、ANMAC 技术评估、签证、长期居留，还是先评估可行性？

## OBA 是否属于可讨论方向
如果你确实有护理教育和护士注册背景，OBA/IQNM 可以作为一个“值得进一步核验”的方向。但现在不能直接说你属于哪个 Stream，也不能说一定能完成注册。

如果只是护理兴趣、没有护理学历，或者只是想快速拿身份，OBA 就不适合作为直接路径。

## 路径拆分
| 层级 | 关键问题 | 当前处理 |
|---|---|---|
| Self-check | 是否进入 IQNM 官方评估入口 | 由本人按官方页面完成 |
| Stream A/B/C | 后续阶段如何分流 | 不能由第三方预判 |
| Portfolio | 学历、注册、身份、翻译和认证材料是否一致 | 先做材料清单 |
| MCQ/NCLEX-RN | 是否需要考试或能否使用已有成绩 | 查 NMBA 当前要求 |
| OSCE | 是否需要赴澳参加临床评估 | 核验地点、时间、费用、签证 |
| Registration | 是否满足 NMBA 注册标准 | 不做承诺 |
| Visa/Employment | 签证和工作是另外两层 | 分开核验 |
| ANMAC | 技术移民评估不是 Ahpra 注册 | 分开核验 |

## 主要风险
- 材料不一致、翻译认证不合规、注册状态不清，会拖慢或影响评估。
- NCLEX-RN 和 OSCE 是考试层，不等于注册完成。
- 注册、就业、签证、ANMAC 和长期居留不能合成一个结果。
- 机构课程最多只能作为服务选择，不能替代 Ahpra/NMBA/ANMAC 官方审核。

## 下一步要核实的问题
- Ahpra/NMBA Self-check 当前说明。
- Portfolio 当前材料要求。
- 当前 English language skills registration standard。
- MCQ/NCLEX-RN、OSCE 和注册申请的当前费用、地点、时间和规则。
- ANMAC、签证、职业清单、州担保或雇主要求是否另有门槛。

## 不建议现在做的事
- 不建议先交培训费或中介费，再确认自己是不是 OBA 目标人群。
- 不建议把“过考试”理解成“能工作/能拿签证/能移民”。
- 不建议用机构口头说法替代官方文件。
```

## English Summary

The skill should treat Australia OBA/IQNM as a registration screening route for internationally qualified nurses or midwives, not as a study-abroad shortcut. It should ask for nursing education, registration status, clinical experience, English, document traceability, budget, OSCE travel feasibility, and final goal. It must separate Ahpra/NMBA registration from ANMAC skilled migration assessment, employment, visa, PR, and immigration outcomes.
